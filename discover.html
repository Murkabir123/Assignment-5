Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML – Gets or sets the HTML content inside an element, including tags. innerText – Gets or sets the visible text inside an element, considering styles. textContent – Gets or sets all text inside an element, ignoring styles.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, you attach a single listener to their common ancestor.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element: document.createElement("tag") Add an element: parent.appendChild(child) or parent.append(child) Remove an element: element.remove() or parent.removeChild(child)
